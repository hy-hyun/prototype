Firebase Console → Firestore Database → Rules에서 다음 규칙으로 변경하세요:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // 임시로 모든 읽기 허용 (개발용)
    match /{document=**} {
      allow read: if true;
      allow write: if false;
    }
  }
}

변경 후 "게시" 버튼을 클릭하여 규칙을 적용하세요.
